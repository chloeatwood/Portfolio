Mobile Applications

Chat n Rate

As part of a team of four, I helped develop Chat n Rate, a mobile app built with Android Studio and Spring Boot that lets users review movies, video games, and books. The app includes real-time chat with friends, the ability to write reviews, leave comments, and like posts. Users can sort and filter reviews, and live updates keep the feed fresh and engaging. This project highlights collaborative full-stack mobile development with real-time features and backend integration.

AR Mobile Mini Game

A mobile version of the last game that is played in Augmented Reality. It was created in Unity and utilizes Vuforia. All the same rules apply in this app as in the previous version except you must play it on your phone. Once you load up the app you must scan an image target after putting it on the floor in order to spawn the room so you can play. You must the use your phone as your eyes in the game to look around. Moving is done by controller buttons on the screen although you may also attempt to walk about in the real world, the game will just be a lot harder that way. Be careful, this game may sound fun but it is a lot more difficult in this version than the last.

Game Development

Dungeon Crawler

I built a terminal-based dungeon crawler in C and C++ that features procedurally generated dungeons, intelligent monster AI, item mechanics, and fog of war—all rendered with ncurses in the command line. A custom min-heap event system manages turn-based gameplay, showcasing object-oriented design and low-level programming. To see how to play it click on the image!

Unity Mini Game

A game I developed so I could learn Unity. The image is the base of the room you spawn in. Everything in the image was created by me in blender. In this game you play as a mouse and need to run around the living room and find all of the cheese before the time runs out. In order to collect all the cheese you must first find the cheese that allows you to jump so you can do a little bit of parkour. Be careful as you can only jump high enough to do the parkour for a limited amount of time before you will have to go collect the cheese again. There is one special piece of cheese that must be collected although you might want to save it for last because if you collect it and do not have enough of the other cheese it will respawn.
